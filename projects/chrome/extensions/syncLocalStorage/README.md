从一个网站获取所有 LocalStorage，并将其设置到另一个网站，即 LocalStorage 同步。

2022-08-04 16:27:09

基于 chrome 版本 103.0.5060.134（正式版本） （64 位）

# 安装

1. chrome://extensions/
2. 开发者模式（右上角）
3. 加载已解压的扩展程序

# 开发调试

2022-08-04 21:35:39

1. 编写扩展
2. 检查 popup.html ，打开 popup.html 的开发者模式，再刷新
   与当前网页并排打开
3. 刷新当前网页

修改扩展之后，需要刷新 popup.html 页面和当前网页。
