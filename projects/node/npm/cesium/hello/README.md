Cesium 搭建场景

# 安装

```sh
pnpm install
```

# 运行

```sh
pnpm run start
```

# 存档

```sh
pnpm run archive
```

## 分发带有依赖的包

1. 解压 master
2. cd master
3. 安装依赖
   npm install --registry=https://registry.npm.taobao.org
4. 测试运行
5. 在 master 文件夹下全选所有文件创建压缩包“master.zip”
6. 重命名压缩包为“某某功能-v0.0.1-孙曙光.zip”
7. 移动并发送压缩包
